# LU BCA Notes

A minimal, high-contrast academic workspace application built for university students using Jetpack Compose.

## Features

- **Dashboard:** Instant access to Notes, PYQs, Calendar, and Time Tables.
- **Offline First:** Dedicated space for downloaded offline content and PDFs.
- **Notices Hub:** Real-time stream of university announcements and circulars.
- **Local Profiles:** Offline workspace management tailored around your current semester and course.

## Tech Stack

- **UI Framework:** Jetpack Compose (100% declarative UI)
- **Design System:** Custom warm-minimalist palette inspired by premium workspaces
- **Language:** Kotlin
- **Architecture:** Uni-directional State Flow (UDF)
