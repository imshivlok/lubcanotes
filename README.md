## Features

- **Unified Dashboard:** Zero-latency routing context maps delivering instant separation between active Notes, PYQs, Calendars, and Time Tables.
- **Offline Infrastructure:** An isolated local storage container engineered to store, serialize, and render cached PDF files completely offline.
- **Notices Hub:** A lightweight dashboard-integrated feed stream dedicated to surfacing official campus announcements and urgent university circulars.
- **Localized Profiles:** Localized on-device data persistence designed to track student states like course tracks and current semester levels to dynamically parse workspace variables.

## Performance Stack

- **UI Framework:** 100% Declarative Jetpack Compose (Eliminating heavy XML layout inheritance strings completely)
- **Aesthetic Core:** High-fidelity, warm-monochrome canvas designed meticulously to eliminate reading fatigue during extended study sessions.
- **Language Layer:** Strict Kotlin code implementation.
- **Data Flow:** Strict Uni-directional Data Flow (UDF) lifecycle architecture.
